Destination

❯
Destination is an navigation application developed by a newly founded company, PostScript.
Destination is an all-in-one application specially targeted for Singaporeans users. Destination includes features such as navigation and various real-time information of different transportation modes. Destination aims to help fellow Singaporeans to save time in travelling and get access to real-time updates on transportation issues.
